# Apollo Music Player

Apollo-Music is a fork from CyanogenMod's <a href="https://github.com/adneal/Apollo-CM">Apollo<a/> music app for Android 4.1+

## What's new

- integrated equalizer
- playlist with the most listened songs
- support for Android 13
